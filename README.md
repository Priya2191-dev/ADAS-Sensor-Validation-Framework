# ADAS Sensor Validation Framework

Automation framework for validating ADAS sensor functionality.

## Features

- Obstacle Detection Simulation
- Collision Warning Validation
- Sensor Data Testing
- Safety Scenario Testing
- Time To Collision Calculation
- Automatic Emergency Braking
- Sensor Fusion
